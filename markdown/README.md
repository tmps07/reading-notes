# 為什麼使用 Markdown

Markdown 的核心價值：

* 易讀
* 易寫
* 易維護
* 可版本控制
* 跨平台
* 可長期保存

適合：

* 閱讀筆記
* 教材整理
* 技術文件
* 教案
* 科展紀錄
* 部落格
* GitHub Pages
* Docsify

## 數學公式

行內：

```md
$a^2+b^2=c^2$
```

顯示：

$a^2+b^2=c^2$

區塊：

```md
$$
E=mc^2
$$
```

顯示：

$$
E=mc^2
$$
