- **「📚 筆耕」**象徵勤於書寫、整理與思考。
- **「🌿 雲笈」**取自古籍《雲笈七籤》，意指典籍、文庫，亦暗合 GitHub Pages 與雲端知識庫的意象。

「筆耕雲笈，編程理科，詩詞並載。以 Markdown 為帙，雲端為藏。既記所學，亦錄所思；既為己用，亦可與人共。願積學以成林，積思以成篇，開卷有益，與眾同欣。」

* Markdown 撰寫
* Docsify 即時渲染
* GitHub Pages 自動部署
* 支援程式碼高亮
* 支援 LaTeX 數學公式
* 主題式分類整理
* 長期知識累積與搜尋
