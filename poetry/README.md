# 古詩詞

> 以閱讀、理解、賞析與整理，重新認識古典文學。

---

## 📚 主題定位

這裡整理：

* 古詩
* 古詞
* 文言文閱讀
* 文學賞析
* 個人閱讀筆記
* 既有創作與作品整理

不是背誦集。

希望透過閱讀與書寫，看見文字背後的時代、情感與思想。

---

# 閱讀方式

建議順序：

① 先閱讀原文
↓
② 理解字詞與背景
↓
③ 賞析結構與意境
↓
④ 記錄心得與聯想
↓
⑤ 延伸閱讀
